# index

This is the first UI endpoint that will be navigated to when your a UI for your app needs to be displayed.

Assuming you set your `index` as shown below:

`{`

`    "name": "Notes",`

`    ....`

`    "bolt": {`

`        "main": "bolt-server.js",`

`        "index": "/notes"`

`    }`

`}`
